# Azure AI Agent Service Documentation

Welcome to the Azure AI Agent Service documentation. This documentation will guide you through setting up, understanding, and working with the Azure AI Agent Service.

## Overview

Azure AI Agent Service is a fully managed service that simplifies the development, deployment, and scaling of AI agents. It provides:

- Automatic Tool Calling
- Secure Data Management
- Pre-built Tool Integration
- Enterprise-Grade Security
- Scalable Infrastructure
